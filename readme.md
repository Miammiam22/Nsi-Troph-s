# Installation des bibliothèques Python

*Ce projet utilise Python. Si vous avez déjà Python 3.14.3 d'installé vous pouvez passer à l'étape suivante.*

## Installer Python

1. Accédez au site d'installation de [Python 3.14.3](https://www.python.org/downloads/release/python-3143/)

2. Allez en bas de la page et cliquez sur la version que votre PC peut éxecuter (*généralement ce sera `Windows installer (64-bit)`*)

## Installer les dépendances

**Ce projet utilise la bibliothèque Ursina et est nécessaire pour lancer le projet**

Depuis un terminal de commande exécutez la commande : ```pip install -r requirements.txt```

# Exécuter le projet

1. Ouvrez un terminal dans la `racine du projet` ou dans `src`

2. Exécutez la commande ```python src/main.py``` ou ```python main.py```
